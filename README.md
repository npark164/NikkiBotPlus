## ğŸ“– About

The WhatsApp Bot assists group admins by providing them with tools to efficiently manage large WhatsApp groups. The bot uses the Baileys library to interact with the WhatsApp Web API and supports multi-device features.

It is lightweight and can be easily customized to add more commands as per your requirements. The bot runs in a Node.js environment and provides QR code-based authentication to link your WhatsApp account.

### General Commands:

- `.help` or `.menu`
- `.tts <text>`
- `.sticker` or `.s`

### Admin Commands:

- `.ban @user`
- `.promote @user`
- `.demote @user`
- `.mute <minutes>`
- `.unmute`
- `.delete` or `.del`
- `.kick @user`
- `.warnings @user`
- `.warn @user`
- `.antilink`

- `.tictactoe @user`
- `.move <position>`

### Group Management:

- `.tagall`

### Other:
